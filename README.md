# SOS Waitlist Page

Minimal, brand-correct waitlist landing page for getsos.app.

## Stack
- Next.js 16+ (App Router)
- TypeScript
- Tailwind CSS
- Satoshi typeface (Google Fonts)

## Design
Single-page waitlist with:
- SOS wordmark (Copper, tracked wide)
- Headline: "The map exists."
- Subhead: "Be first to hold it."
- Email input + submit button
- Redirect to getsos.beehiiv.com on submit

## Colors
- Void: `#0E0C1E`
- Copper: `#C9A27A`
- Near-White: `#F4EFE8`
- Warm Grey: `#9A948C`
- Surface: `#161422`

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

```bash
vercel
```

Or connect the GitHub repo to Vercel directly via https://vercel.com/new.
